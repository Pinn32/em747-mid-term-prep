description: Which quotation marks to use when editing Mid-term CheatSheet.md
type: feedback

Use curly/typographic quotes, not straight quotes, when editing this file.

- Prefer `"..."` `"..."` (straight double quotes) first
- Fall back to `'...'` `'...'` (straight single quotes)
- Never use `“...”` (curly double quotes)

**Why:** User's explicit style preference for this Obsidian cheatsheet file.
**How to apply:** Any time text content (not code blocks) requires quotation marks in Mid-term CheatSheet.md, use curly quotes.
